# 如何获取 [Amp](https://ampcode.com) 的 系统提示词

1. 使用 VScode 登录 Amp
2. 问题 短接 查询 到 Amp
3. 按住 Alt (Windows) 或 Option (macOS) 并单击工作区按钮

![](./view-thread-yaml.png)

4.点击查看线程YAML

# 注释

Amp 使用的 系统提示词 已调整为 Sonnet 4.x，并将其他 LLM 注册为工具（“oracle”）。要获得 `GPT-5` 调整后的 系统提示词，您需要使用以下内容配置 VSCode 用户 设置，然后再次按照上述步骤操作
